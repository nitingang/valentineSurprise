# Valentine Surprise — Single-file webpage

Open `index.html` in a browser, or host with GitHub Pages.

## How to use
1. Edit `index.html` (optional) — change the `SPIN_OPTIONS` array near the top of the script to customize the spinner choices.
2. Commit and push to a GitHub repository.
3. Enable GitHub Pages on the repository (see steps below).
